# Rustfmt
Project : https://github.com/rust-lang/rustfmt 

#### Installation
1. Install the nightly toolchain if necessary with ```rustup install nightly```
2. Add rustfmt : ```rustup component add rustfmt --toolchain nightly```

#### Usage
```cargo +nightly fmt```
